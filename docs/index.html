Cascading Prompt Sheets
CPS Specification

Uma especificação aberta para governança de IA por proximidade. CPS define um sistema de arquivos .prompt que estabelecem regras de governança para geração de código assistida por IA.

Especificação Técnica

1. Arquivo .prompt

Arquivos de configuração que definem regras de governança para componentes específicos.

Button.prompt SYNTAX v1 
[component: action-button]
[context: e-commerce-checkout]

@governance
tone: "urgente e direto"
restricted-terms: ["barato", "grátis"]
accessibility-check: "sempre validar aria-label"

@logic-constraints
max-characters: 30
dynamic-state: "se loading, exibir spinner"

Metadados

  • [component] - Identificador do componente
  • [context] - Contexto de aplicação

Seções

  • @governance - Regras de governança
  • @logic-constraints - Restrições lógicas

2. Sistema de Cascata

Herança de regras baseada em hierarquia de diretórios, similar ao CSS.

1

Herança Global

Arquivo global.prompt na raiz é herdado por todos os componentes.

2

Especificidade Local

Regras em Component.prompt sobrescrevem regras globais.

3

Resolução

Prompt Resolver combina todas as regras aplicáveis em um prompt final.
