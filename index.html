Cascading Prompt Sheets (CPS) - Uma camada de governança componentizada para IA inspirada na estrutura de cascata do CSS. Onde cada componente tem suas próprias regras.

Ver Documentação GitHub Repository

QUICK START

# Clone o projeto
git clone https://github.com/mikaelcarrara/prompt-sheet.git
cd prompt-sheet

# Instale dependências
npm install

# Inicie o servidor de demonstração
npm start

Como Funciona

CPS implementa governança por proximidade através de arquivos .prompt colocalizados

1. Herança em Cascata

Regras definidas na raiz (ex: global.prompt) são herdadas por todos os subdiretórios. 

global.prompt
@governance
tone: "profissional"
accessibility: "sempre"

2. Especificidade Local

Regras locais (ex: Button.prompt) sobrescrevem regras globais. 

Button.prompt
@governance
tone: "urgente"
max-chars: 30

3. Injeção de Contexto

Prompt Resolver lê a árvore do componente e injeta as regras no prompt de sistema da IA. 

const resolver = new PromptResolver();
const prompt = await resolver.generatePrompt(
  'components/Button', 
  'Crie um botão...'
);
